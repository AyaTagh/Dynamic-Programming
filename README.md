# Dynamic Programming – LCS Analysis

Academic project on Dynamic Programming using the Longest Common Subsequence (LCS) problem.

## Overview

Comparison between a naïve recursive algorithm and an optimized dynamic programming solution through:

* complexity analysis,
* C implementation,
* experimental performance evaluation.

## Concepts & Skills

Dynamic Programming • Algorithms • Complexity Analysis • C Programming • Performance Benchmarking

## Main Result

Experimental results confirmed the theoretical complexity gap between exponential recursion and polynomial dynamic programming.
